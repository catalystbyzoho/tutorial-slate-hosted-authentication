# tutorial-slate-hosted-authentication

A simple web application built using Catalyst Slate and Catalyst Hosted Authentication to demonstrate user authentication. It allows users to sign up, log in, and reset passwords using Catalyst’s hosted authentication, and the application is hosted on Catalyst Slate.

Kindly refer to the help documentation for Catalyst Slate and Catalyst Authentication below:

Catalyst Slate – https://docs.catalyst.zoho.com/en/slate/help/introduction/

Catalyst Authentication – https://docs.catalyst.zoho.com/en/cloud-scale/help/authentication/introduction/
